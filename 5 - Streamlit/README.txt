STREAMLIT

La présentation est basée sur le fichier :
- 01_Homepage.py
Il s'agit du fichier Python principal pour lancer le streamlit.

Pour lancer le streamlit, il convient de suivre les étapes suivantes :
1 - Cloner le répertoire en local ;
2 - Installer les versions des packages Python nécessaires en lançant la commande pip install -r requirements.txt
3 - Lancer le streamlit via la commande suivante sur le terminal : streamlit run 01_Homepage.py
